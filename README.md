# Reminder_Calendar

알람을 설정해두면 정해진 시간에, 음악이 재생되는 리마인더 프로그램

개발방향
----------------
다양한 정보를 한눈에 쉽게 알아볼 수 있도록, 가독성과 간결성을 중요시 하였다.
